# res
Simple CLI tool to generate images from given resolution and color

### Usage Example
    python res.py 300x200:darkgreen 200x200:red

### Format
    300x200:darkgreen (WIDTHxHEIGHT:COLOR)
    or
    300x200 (WIDTHxHEIGHT) : Color defaults to orange

### Color

- HTML Colors:
    
      red, black, lightblue, ...

- HEX Colors:

      #eee, #eeeeee, ...
